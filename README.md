# 🛡️ Smart AI Based Campus Safety System

## 📌 Overview
The Smart AI Based Campus Safety System is a Flask-based web application developed to improve campus security using AI concepts.

## 🚀 Features
- 🏠 Home Page
- 📹 Live Camera
- 🚨 Emergency SOS
- 📊 Dashboard
- 📜 Alert History
- 📍 Location
- 📞 Emergency Contacts
- 🔐 Login System

## 🛠 Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- OpenCV
- Git
- GitHub

## ▶️ How to Run

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
